# Brand Pulse Monitor

An end-to-end, free, open-source project for monitoring brand sentiment and customer pulse.

## Features (planned)

- Ingest customer feedback (reviews, comments, survey text)
- Run open-source sentiment analysis
- Detect topics / themes using embeddings and clustering
- Visualize trends in a Streamlit dashboard
- Automate data refresh via GitHub Actions

## Tech stack

- Python
- Hugging Face models (sentiment, embeddings)
- Streamlit for the dashboard
- GitHub Actions for automation
